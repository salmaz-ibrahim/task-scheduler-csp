# 🧠 Task Scheduler (CSP)
![Python](https://img.shields.io/badge/Python-3.10-blue)
![AI](https://img.shields.io/badge/AI-CSP-orange)
![Status](https://img.shields.io/badge/Status-Ongoing-yellow)
A Python-based task scheduler using Constraint Satisfaction Problem (CSP) principles.

## 🚀 Features
- Assign tasks to time slots without conflicts
- Input validation (empty, duplicates, invalid format)
- Automated test suite

## 🧪 Testing
Run:

*python tests.py*

## 🧠 Concepts
- Constraint Satisfaction Problems (CSP)
- Set-based constraint enforcement
- Input validation and edge case handling

## 🚧 Future Improvements

- Integrate AI (Gemini) to dynamically generate time slots  
- Connect to Google Calendar API for automatic scheduling  
- Support real-time user input and time parsing
